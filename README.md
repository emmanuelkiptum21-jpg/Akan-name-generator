# Akan Name Generator

## Description
This is a simple web application that calculates the day of the week a user was born and displays their Akan name based on their gender. Akan names come from Ghanaian culture.

---

## Author
Emmanuel Kipkoech

---

## How to Use
1. Open the project folder in VS Code  
2. Open `index.html` in a web browser  
3. Enter your day, month, and year of birth  
4. Select your gender  
5. Click the **Generate Name** button  

---

## Features
- User input validation
- Calculates the day of the week using a formula
- Displays the correct Akan name
- Simple and clean user interface
